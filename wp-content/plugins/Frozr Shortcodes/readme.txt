== Description ==

A set of shortcodes spicially created for Frozr WP theme.

== Installation ==

1. Upload `frozr-shortcodes.php` to the `/wp-content/plugins/` directory of your WordPress installation
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Success! You're ready to start adding shortcodes to your blog posts and pages.
NOTE: The plugin requires Frozr WP theme to be activated to work.

For usage instructions and feedback please visit http://alamento.dokkanplus.com/?p=1891
