=== Plugin Name ===
Requires at least: 4.4
Tested up to: 4.5.3
Stable tag: 4.4

Lazy Eater, a multi-vendor restaurants system.

== Description ==

Lazy Eater is an Frozr WP-Theme extension that turns Frozr to a fully featured multi-vendor online food ordering system.
Get support at: http://www,mahmudhamid.com
