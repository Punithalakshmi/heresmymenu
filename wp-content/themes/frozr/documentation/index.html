B) How to Install Frozr - top

Installing the theme is simple like installing any other WordPress Theme.

  1. Upload the Frozr.zip file to your WordPress Themes directory.
  2. In your WordPress back-end, go to Appearance > Themes and activate Frozr.
  3. Enjoy features through the WordPress Customizer.
